# nba-data-analysis
End-to-end NBA data pipeline and performance analysis using Python and R

NBA Data Pipeline & Analysis

Overview
This project collects and analyzes NBA player performance data from 1996–2023.

Features

Python script to collect data from NBA stats API

Automated multi-season data collection

Data cleaning and structuring

Performance analysis using statistical methods

Tools Used

Python (requests, pandas)

R (tidyverse, ggplot2)

Key Insight

Usage metrics (minutes, shot attempts) strongly correlate with performance

Player performance trends change significantly with age

Purpose
This project demonstrates data collection, cleaning, and analysis workflows similar to real-world data analyst roles.
